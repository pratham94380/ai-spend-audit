# Pricing Notes

Enterprise and API pricing vary significantly based on:
- contract size
- token usage
- seats
- enterprise agreements

For MVP purposes, approximate values were used where official public pricing was unavailable.

The audit engine is designed as a recommendation and optimization tool rather than an official billing calculator.
